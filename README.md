# Payment Options Display - WordPress Custom Plugin

A custom WordPress plugin that dynamically displays various **payment options** and **installment plans** for online stores (WooCommerce or any custom store).

![Plugin Preview](screenshot/view.png)

## ✨ Features

- Dynamically shows multiple payment methods
- Supports installment / EMI options
- Clean and modern design
- Easy to customize (colors, text, payment icons)
- Lightweight and fast loading
- Works with WooCommerce and custom themes

## 🛠️ Technologies Used

- WordPress
- PHP
- JavaScript
- CSS
